# tools-lib

## Getting Started

Init library

```bash
$ npm run init
```

Publish library

```bash
$ npm run pub
```

Install dependencies,

```bash
$ yarn install
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
